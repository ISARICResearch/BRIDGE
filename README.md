# BRIDGE

ISARIC BRIDGE is a web-based application designed to operationalize [ISARIC ARC](https://github.com/ISARICResearch/ARC) and edit any ISARIC CRF, tailoring it to specific outbreak contexts. By selecting and customizing clinical questions and ensuring necessary data points for each, BRIDGE automates the creation of Case Report Forms (CRFs). It generates the data dictionary and XML needed to create a REDCap database in the ARC structure and produces paper-like CRFs and completion guides.

## About BRIDGE

BRIDGE (BioResearch Integrated Data tool GEnerator) is a resource for researchers and healthcare professionals involved in outbreak and emerging public health research.

### Key Features

- **Multilingual support (AI-translated):** Generate CRFs in **English, Spanish, Portuguese, and French**.  
  > *Note: AI-generated translations may require local review before deployment.*

- **Choose Questions:** Navigate ARC as a tree (version → forms → sections → questions) to select relevant variables. Start with pre-defined Templates for quicker CRF creation.

- **Customize CRFs:** Tailor CRFs with selectable measurement units and checkable answer options. Ensure relevance and precision for site- or disease-specific studies.

- **Save and Share Progress:** Save partially completed CRFs as templates to continue later or share with collaborators. This enables consistent teamwork and reduces duplication.

- **Capture:** Generate REDCap-ready files including the data dictionary, XML configuration, paper-like CRFs, and completion guides.

- **Paper-like Design:** Professionally formatted CRFs suitable for both digital and printed use.

- **Rapid Adaptation for Outbreaks:** Quickly customize CRFs to meet the needs of emerging or re-emerging diseases.

## BRIDGE 1.2 Highlights

- **Optimized Interface:** Cleaner design and improved navigation for faster workflow.  
- **Performance Improvements:** Faster CRF generation and ARC data processing.  
- **Bug Fixes & Reliability Enhancements:** Including fixes for focused cell merges, list ordering, and image URLs.  
- **New Features & Enhancements:**  
  - Pregnancy form  
  - ARC units  
  - Caching of CRF/ARC schema calls  
  - Pre-commit configuration  
  - UI improvements  

These build on core capabilities: selecting ARC variables, applying templates, generating Clinical Characterization XML, REDCap Data Dictionaries, and paper-like CRFs.

## Files Generated by BRIDGE

- **Clinical Characterization XML:** Recommended configuration and structure for studies.  
- **REDCap Data Dictionary:** CSV ready for REDCap database creation.  
- **Paper-like CRF:** PDF suitable for field deployment, available in multiple languages (AI-translated).  
- **Completion Guide:** Detailed PDF with variable definitions and instructions for standardized data collection.
